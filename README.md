## 🎯 Overview

This application provides an end-to-end solution for:

1. **Loan Rejection Prediction**: Machine learning models to predict loan rejection risk
2. **Risk Analytics**: Comprehensive analysis of financial risk factors
3. **Interactive Dashboard**: Real-time data visualization and exploration
4. **Predictive Tool**: Easy-to-use interface for individual loan assessment

## ✨ Features

### 📊 Dataset Overview
- Complete data preview with filtering options
- Column information and data types
- Missing value analysis
- Summary statistics

### 📈 Risk Analytics
- Risk indicator analysis (DTI, Leverage, Savings)
- Correlation matrix visualization
- Risk score calculation
- Demographic risk analysis
- Interactive charts and graphs

### 🤖 Model Performance
- Multiple model comparison
- Feature importance analysis
- ROC curves and performance metrics
- Confusion matrix visualization
- Classification report

### 🎯 Prediction Tool
- Interactive input form
- Real-time prediction
- Risk factor identification
- Financial metrics calculation
- Probability scoring
